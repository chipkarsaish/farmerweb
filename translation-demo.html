üìù How to Use Translation System

  1. Add data-i18n="key.path" attribute to any HTML element
  2. The key path corresponds to the JSON structure in translation files
  3. Click language buttons above to see translations in action
  4. Language preference is saved in localStorage
  5. Works completely offline after initial load
